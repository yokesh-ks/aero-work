# Version 1.1.0 - 2026-01-17

## Projects Backend API

### Added

#### Database Layer
- SQLite database integration using `better-sqlite3` with WAL mode for performance
- `projects` table with full schema for GitHub-linked project management
- Database initialization on app startup with automatic table/index creation
- Graceful database connection cleanup on app quit

#### Project Management API
- `projects:list` - List all projects with optional filtering (search, owner, status, recent)
- `projects:getById` - Get single project by UUID
- `projects:getByGitHubRepoId` - Get project by GitHub repository ID
- `projects:create` - Create new project with validation
- `projects:updateStatus` - Update project status (linked/cloning/error)
- `projects:updateLastOpened` - Track project access timestamps
- `projects:delete` - Remove project from database
- `projects:existsByGitHubRepoId` - Check for duplicate projects

#### GitHub Integration API
- `github:storeToken` - Secure token storage using Electron's safeStorage (OS-level encryption)
- `github:getToken` - Retrieve stored token
- `github:clearToken` - Logout/remove stored token
- `github:isAuthenticated` - Validate authentication status
- `github:getAuthState` - Get auth state with user info (login, avatar, name)
- `github:openOAuthUrl` - Launch OAuth flow in browser
- `github:getRepositories` - Fetch all accessible repositories (owned, collaborator, org)
- `github:getRepository` - Fetch specific repository by owner/name
- `github:validateScopes` - Validate token has required OAuth scopes

#### Git Operations API
- `git:clone` - Clone GitHub repository to local directory
- `git:validateRemote` - Validate local repo's remote matches expected GitHub URL
- `git:isRepository` - Check if directory is a Git repository
- `git:getCurrentBranch` - Get current branch name
- `git:getStatus` - Get repo status (clean, ahead/behind, tracking)
- `git:pull` - Pull latest changes from remote
- `git:directoryExists` - Check directory existence
- `git:isDirectoryWritable` - Check write permissions
- `git:isDirectoryEmpty` - Check if directory is empty

#### Dialog API
- `dialog:selectDirectory` - Native directory picker with create folder option
- `dialog:showMessage` - Customizable message box (info, warning, question, error)
- `dialog:showError` - Native error dialog

#### Architecture
- IPC handlers in main process with standardized `ApiResponse<T>` format
- Preload script exposing typed `window.electronAPI` via contextBridge
- Renderer services for simplified API consumption
- Full TypeScript type definitions for all entities and operations

### Documentation
- API documentation in `api-collections/projects/`
- Testing guide with DevTools console examples
- Test utilities script for interactive API testing

### Dependencies Added
- `better-sqlite3` - SQLite database
- `simple-git` - Git operations
- `@octokit/rest` - GitHub API client
- `uuid` - UUID generation
- `@electron/rebuild` - Native module rebuilding

### Configuration
- Vite externalization for native modules (`better-sqlite3`, `simple-git`)
- Automatic native module rebuild via `postinstall` script
