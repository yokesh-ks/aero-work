# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in this project, please do not open a public issue. Instead, please email us at security@aero-work.dev.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Security Updates

We will patch security vulnerabilities as soon as possible and release updates. We recommend keeping your dependencies up to date.

Thank you for helping us keep Aero Work safe!
